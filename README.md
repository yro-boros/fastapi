# FastAPI Book API

Простой REST API для управления списком книг с использованием FastAPI.

## Установка

Установите необходимые библиотеки:

```bash
pip install fastapi



Запуск
Запустите приложение:

Bash
python Fast_API_1.py
Сервер будет доступен по адресу: http://127.0.0.1:8000

API Эндпоинты
Получить все книги
GET /items

Возвращает список всех книг.

Получить книгу по ID
GET /items/{item_id}

Возвращает информацию о конкретной книге по её ID. Если книга не найдена, возвращается ошибка 404.

Добавить новую книгу
POST /items

Добавляет новую книгу в список.

Тело запроса (JSON):

Json

Применить
{
  "title": "Название книги",
  "author": "Автор книги"
}
